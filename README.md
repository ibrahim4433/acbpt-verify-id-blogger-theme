# acbpt-verify-id-blogger-theme
a custom, fully functional Blogger XML theme designed strictly for a "Certificate Verification" portal.

## Blogger quick setup

1. Go to **Blogger → Theme → Customize → Backup/Restore → Upload** and upload the repository `theme.xml` file.
2. Go to **Layout** and open the **Header (AmericancanadianBoard Logo)** gadget.
3. The theme now keeps a logo visible in the header at all times. If needed, replace it using a direct image URL in the Header gadget settings.
4. Save and preview. The page is styled with an ACBPT professional navy/red palette for certificate verification.
