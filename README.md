# acbpt-verify-id-blogger-theme
a custom, fully functional Blogger XML theme designed strictly for a "Certificate Verification" portal.

## Blogger quick setup

1. Go to **Blogger → Theme → Customize → Backup/Restore → Upload** and upload `/home/runner/work/acbpt-verify-id-blogger-theme/acbpt-verify-id-blogger-theme/theme.xml`.
2. Go to **Layout** and open the **Header (ACBPT Logo)** gadget.
3. Keep the default logo or replace it with your official ACBPT logo URL/image.
4. Save and preview. The page is styled with an ACBPT professional navy/red palette for certificate verification.
