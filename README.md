# acbpt-verify-id-blogger-theme

Professional Blogger XML theme for a Certificate Verification Portal.

## Blogger quick setup

1. Go to **Blogger → Theme → Customize → Backup/Restore → Upload** and upload `theme.xml`.
2. Go to **Layout** and edit the **Header** gadget (`Header1`) to upload/change the portal logo dynamically.
3. Create certificate posts where the **post title is the Student ID** and the **post body contains the certificate image/content**.
4. Users search by ID from the portal search box (`/search?q=...`) and matching post results are shown in the results area.
